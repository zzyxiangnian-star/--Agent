# 贡献者行为准则

## 我们的承诺

我们承诺为所有参与者提供友善、包容且无骚扰的协作环境，不因年龄、体型、残障、族群、性别特征、性别认同与表达、经验水平、教育程度、社会经济状况、国籍、个人外貌、种族、宗教或性取向而区别对待。

## 行为标准

有助于营造积极环境的行为包括：

- 使用友善、包容的语言并尊重不同观点。
- 接受建设性反馈，对社区负责并关注共同目标。
- 在分歧中保持专业，避免人身攻击、骚扰与公开他人隐私信息。

不可接受的行为包括侮辱、威胁、性化言论或图像、持续打扰，以及其他在专业环境中不适当的行为。

## 执行责任

项目维护者负责解释并执行本准则，可以删除不当评论、提交、代码或其他贡献，并可临时或永久禁止行为不当者参与。

## 适用范围与报告

本准则适用于项目空间及代表项目参与公共活动的场合。需要私下报告的事件，可通过 GitHub 仓库的安全报告渠道联系维护者。维护者将尊重报告者的隐私并作出适当回应。

## 来源

本准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html) 编写。
